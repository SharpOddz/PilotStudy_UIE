# PilotStudy_UIE

Because a real public underwater swimmer-video dataset with extensive pose labels is not readily available, the pilot study uses SwimXYZ as a controlled swimming-pose dataset with ground-truth 2D/3D joints. Each frame is processed using several representative UIE methods, and the resulting videos are evaluated using both conventional image-quality metrics and downstream pose-estimation metrics. The goal is not to claim real-world generalization from synthetic data alone, but to test whether UIE method rankings differ between perceptual image quality and swimmer-pose reliability.
